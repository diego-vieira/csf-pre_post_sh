# ConfigServer Security & Firewall (CSF) - csfpre.sh / csfpost.sh

## Install
Installation is quite straightforward:

```bash
cd /usr/local/src
git clone https://github.com/diego-vieira/csf-pre_post_sh.git
cd csf-pre_post_sh
sh install.sh
```
